Context Sync

Sync Microsoft 365 into your knowledge graph

Automatic daily work context extraction from Outlook, Teams, and To Do. Built for GTD practitioners who want their digital ecosystem organized.

View on GitHub Get Started

Features

📧

Email Context

Extract flagged emails and important conversations from Outlook. Never lose track of follow-ups.

✅

Task Sync

Pull Microsoft To Do tasks into your daily notes. See what's actionable at a glance.

💬

Teams Activity

Capture important chat threads and mentions. Keep context from scattered conversations.

📅

Calendar Integration

See your day's meetings with attendee context. Prep for conversations automatically.

🗂️

GTD-Ready

Structured output for projects, next actions, waiting-for, and someday/maybe lists.

🌐

Knowledge Graph

Markdown output integrates with Obsidian, Logseq, or any Zettelkasten system.

Installation

Quick Start

# Clone the repository git clone https://github.com/Midtown-Technology-Group/context-sync.git # Install dependencies pip install -r requirements.txt # Configure python -m context_sync setup

Azure Deploy

# Deploy to Azure Functions ./deploy.ps1 -Environment production # Or use the Azure portal template # See azure-deployment.json
